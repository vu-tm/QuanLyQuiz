# QuanLyQuiz

### Vào http://localhost/phpmyadmin/ → `New` → Database name = `quiz`  → `Create`

### Vào `quiz` → `Import` → `Choose File` → QuanLyQuiz\database\quiz.sql → `Import`

### Vào Netbeans → chuột phải project QuanLyQuiz → `Clean and Build` 

### Vào GUI → Main.java → chuột phải chọn `Run File`
